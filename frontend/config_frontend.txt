Pasos para la Configuración Inicial del Frontend (US-29)

1. Navegar al Directorio del Frontend Abre una terminal y sitúate en la carpeta frontend de tu proyecto.
cd frontend

2. Inicializar el Proyecto con Vite y React Usa Vite para crear un nuevo proyecto de React en el directorio actual.
npm create vite@latest . -- --template react

3. Instalar Dependencias del Proyecto Instala todas las librerías necesarias para el frontend, según lo especificado en herramientas.md y requirements_frontend.txt.
npm install axios react-router-dom @tanstack/react-query zustand react-hot-toast react-hook-form @headlessui/react @heroicons/react clsx recharts

4. Instalar y Configurar TailwindCSS Instala las dependencias de desarrollo para TailwindCSS y genera los archivos de configuración.
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
Esto creará los archivos tailwind.config.js y postcss.config.js.

5. Configurar las Rutas de Plantillas de Tailwind Abre el archivo tailwind.config.js y configura la propiedad content para que TailwindCSS analice tus archivos de componentes y páginas en busca de clases.
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

6. Añadir las Directivas de Tailwind al CSS Abre el archivo src/index.css, elimina todo el contenido existente y añade las directivas de Tailwind.
@tailwind base;
@tailwind components;
@tailwind utilities;

7. Configurar el Archivo de Entorno (.env) Crea un archivo .env en la raíz del directorio frontend para definir la URL base de tu API de backend.
VITE_API_URL=http://localhost:3000/api
Nota: Vite expone las variables de entorno con el prefijo VITE_ al código del cliente.

8. Ejecutar el Servidor de Desarrollo El package.json generado por Vite ya incluye los scripts necesarios. Puedes iniciar el servidor de desarrollo con:

npm run dev
