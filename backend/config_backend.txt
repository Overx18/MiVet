Pasos para la Configuración Inicial del Backend (US-28)
1. Navegar al Directorio del Backend Abre una terminal y sitúate en la carpeta backend de tu proyecto.
cd backend

2. Inicializar el Proyecto Node.js Crea un archivo package.json para gestionar las dependencias.
npm init -y

3. Instalar Dependencias de Producción Instala los paquetes necesarios para que la aplicación funcione, según se especifica en herramientas.md.
npm install express sequelize mysql2 dotenv helmet cors jsonwebtoken bcryptjs passport express-validator http-errors uuid nodemailer

4. Instalar Dependencias de Desarrollo Instala nodemon para que el servidor se reinicie automáticamente al detectar cambios en los archivos.
npm install -D nodemon

5. Configurar el Archivo de Entorno (.env) Crea el archivo .env en la raíz del directorio backend y añade las variables para la conexión a tu base de datos MySQL y el secreto para los tokens JWT. Reemplaza los valores de ejemplo con tus credenciales.
# Base de Datos
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=tu_contraseña_secreta
DB_NAME=mivet_db
DB_PORT=3306

# JWT
JWT_SECRET=un_secreto_muy_seguro_para_jwt

6. Crear la Base de Datos Asegúrate de tener un servidor MySQL en ejecución y crea la base de datos mivet_db (o el nombre que hayas configurado en tu archivo .env).

7. Configurar Scripts en package.json Añade un script en tu package.json para iniciar el servidor en modo de desarrollo usando nodemon.
"scripts": {
  "dev": "nodemon src/app.js",
  "start": "node src/app.js"
}
